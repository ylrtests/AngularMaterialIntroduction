Define if an element spans more than one column/row using columnspan/rowspan attributes

Gutter size property to define this space

Default: Height of row is equal to width of column, to override this we use the rowHeight attribute

  • Using rowHeight: 100px, defines static value
  • Using rowHeight: 2:1 aspects ratio, the height is half of a column's width
  • Using rowHeight: fit and defining a height for the container, fits automatically
